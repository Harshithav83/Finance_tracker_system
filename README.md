# Finance Tracker System

A complete finance tracking system with Django backend and responsive frontend.

## Features

### Backend
- Add, view, edit, delete transactions
- Filter transactions by date, category, type, amount
- Role-based access (Viewer, Analyst, Admin)
- REST API endpoints

### Frontend
- Responsive dashboard with financial cards
- Real-time balance calculation
- Interactive charts (Category breakdown, Monthly trends)
- Transaction table with pagination
- Filter form for advanced search
- Add/Edit forms with validation
- Delete confirmation modal
- Mobile-friendly design

## Tech Stack

### Backend
- Django 5.2
- MySQL
- Django ORM
- Django Authentication

### Frontend
- HTML5
- CSS3 (Custom styles + Bootstrap 5)
- JavaScript
- Font Awesome icons
